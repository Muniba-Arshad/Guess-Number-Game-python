# Guess-Number-Game-python
A simple Guess Number Game built with python using while loop, if-elif-else, user input, and random module.

# Guess Number Game
A beginner Python project.

## Description
The computer generates a random number between 1 and 100.
The player has three attempts to guess the correct number.
The game also gives hints like:
- Too high
- Too low
